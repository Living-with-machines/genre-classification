We create a tokenizer and a model using the pre-trained model we created.

tokenizer = AutoTokenizer.from_pretrained("davanstrien/bl-books-genre")
model = AutoModelForSequenceClassification.from_pretrained("davanstrien/bl-books-genre")

To make the process of doing inference very straightforward we can use a pipeline. These are intended to make the process of using models for inference easy for a wide range of tasks. We need to tell the pipeline what kind of task weâ€™re doing and pass in our model and tokenizer.

Weâ€™ll create a function to try and get the prediction. It does a quick check to see if we need to chop off some of our text (our model has a max len of 512).

import numpy as np

def try_predict(x):
    pred = np.nan
    if len(x) > 512:
        x = x[:512]
    try:
        pred = classifier(x)
    except Exception as e:
        print("\U0001F92E", e, x)
        pass
    return pred

This is quite an ugly solution but since weâ€™re not planning to do anything beyond make these predictions once weâ€™ll allow a bit of sloppiness ðŸ˜‰. We can use progress_apply to get a little progress bar so we know how long itâ€™s going to take.

We now just do a bit of tidying to get our labels into the format we want

df["predicted_label"] = preds.apply(lambda x: x[0]["label"])

df["predicted_label"][0]

'Fiction'

df["prob"] = preds.apply(lambda x: x[0]["score"])

def get_fiction_prob(x):
    if x.predicted_label == "Fiction":
        return x.prob
    else:
        return 1 - x.prob

df["fiction_probs"] = df.apply(get_fiction_prob, axis=1)

df["non_fiction_probs"] = 1 - df["fiction_probs"]

df = df.drop(columns=["prob"])
